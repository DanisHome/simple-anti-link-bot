# Anti Link Discord Bot 🚀

## About the Bot 📝
This Bot blocks Links from your Discord Server. Its very helpfull if you have a big Community!

## Installation 💾
1. Open Console
2. Type in: pip install -r requirements.txt
3. After that type: py main.py
